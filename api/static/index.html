AnalogDB API

Welcome to the AnalogDB, a free resource providing analog photos.

All film pictures are scrapped from r/analog. This API is open-source and available on github. All endpoints are public and use a RESTful JSON interface. Project is currently under development and subject to change

Please use the following URI to access the endpoints: https://analog-go.herokuapp.com

Available Methods

All endpoints take an optional {count} parameter. This determines the number of posts to return (defaults to 1).

Get latest pictures

curl https://analog-go.herokuapp.com/latest/{count}

Get top pictures

curl https://analog-go.herokuapp.com/top/{count}

Get random pictures

curl https://analog-go.herokuapp.com/random/{count}
